## 📌 Sobre este módulo

Este é o primeiro módulo da pós-graduação em **Engenharia de Software com IA Aplicada**. Aqui começa minha jornada para aprender a usar Inteligência Artificial no dia a dia do desenvolvimento de software, com foco total em **prática e resultado real**.

---

## 🎯 O que aprendi

- ✅ A proposta do curso: IA aplicada para **desenvolvedores**, não cientistas de dados
- ✅ Por que **JavaScript** é a linguagem escolhida — roda IA direto no navegador
- ✅ Como tirar o máximo do curso: praticar, adaptar exemplos, criar projetos próprios
- ✅ A importância de publicar projetos no **GitHub** como portfólio profissional
- ✅ Como usar a **comunidade no Discord** para aprendizado colaborativo
- ✅ Visão geral de todos os módulos que vêm pela frente:

| Módulo | Tema |
|--------|------|
| 02 | Machine Learning, Deep Learning e IA |
| 03 | Sistemas de Recomendação |
| 04 | Web ML — Vencendo Qualquer Jogo |
| 05 | IA na Web, LLMs e Algoritmos Genéticos |
| 06 | Prompt Engineering |
| 07 | Ferramentas de IA para Devs |
| 08 | MCPs e Automação |
| 09 | Modelos Open-Source vs Proprietários |
| 10 | RAG, Embeddings e Busca Semântica |
